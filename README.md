# Interview AI

Welcome to **Interview AI**! I built this project to help candidates prepare for their interviews using advanced artificial intelligence.

## About the Project

Interview AI is a modern web application that generates personalized interview reports, complete with technical and behavioral questions, tailored roadmaps, and match scores based on real job descriptions and resumes.

### Features
- **AI-Powered Insights**: Generate tailored interview reports and comprehensive preparation roadmaps.
- **Modern UI/UX**: Designed with a sleek, premium dark mode aesthetic and responsive, intuitive layouts.
- **Fast Performance**: Built using React and Vite for lightning-fast development and optimized production builds.

## Getting Started

To run this project locally, simply follow these steps:

1. Clone the repository to your local machine.
2. Navigate to the project directory and install the dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
